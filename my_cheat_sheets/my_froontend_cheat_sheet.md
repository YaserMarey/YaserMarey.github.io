# My Frontend Cheat Sheet

## Install Node
```
curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -
sudo apt-get install -y nodejs
```
### delete all spec.ts files from exisiting angular project on Linux
```
sudo find . -type f -name '*.spec.ts' -delete
```
